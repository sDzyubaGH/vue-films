## Требования

Перед запуском необходимо установить (если еще не установлено):

1. Docker Engine / Docker Desktop
2. make (если умеете пользваться docker, то он не нужен)

Информацию об установке docker можно найти на [официальном сайте](https://docs.docker.com/desktop/).
Все дальнейшие команды следует выполнять из корневой директории проекта (там где лежит docker-compose.yml)

## Содержимое

```bash
interview-postgres
├── docker-compose.yml      # файл для автоматизации развертывания
├── Dockerfile              # файл с командами по сборке образа
├── Makefile                # файл для упрощения взаимодействия пользователя
├── preparations            # директория с подготовительными скриптами
│   ├── 01-fill.sql         # заполнение данными
│   └── 02-create-user.sql  # создание пользователя с правами на SELECT
└── README.md               # этот файл
```

## Запуск

Для запуска выполните следующие команды:

```bash
make up # с помощью make
# ИЛИ
docker compose up -d --build # без make
```

## Отладка

Для просмотра логов postgres можно воспользоваться следующими командами:

```bash
make logs # с помощью make
# ИЛИ
docker compose logs postgres # без make
```

## Чистка

После конца работы с БД можно воспользоваться слеюущими командами для очистки системы:

```bash
# С помощью make
make down
make clean

# ИЛИ

# Без make
docker compose down
docker volume rm interview-postgres_postgres
docker image rm interview-postgres-postgres
```
